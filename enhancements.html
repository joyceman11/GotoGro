Types of enhancements used on this site

Responsive Design:

Having a responsive layout allows different users to view your webpage on different devices with the layout of the page adjusted to suit that device In order to achieve this I have included 'viewport' in the html documents in the head section to indicate that this page will be responsive to the device it is being viewed on. Not only does it only adjust text displayed on the screen it also adjusts images as well.

These adjustments are suitable for different media displays listed below:
  • - Medium Displays: Tablets, iPads
  • - Small Displays: Phones

Even if you weren't on another device, you are able to see this through the Firefox or Chrome browser, by selecting 'Developer Tools' and choosing the device you want to view it from.
